# setup and deploy

```
$ npm i
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
$ ./deploy.sh
```

# build container locally

```
./build.sh
```

# run container locally

```
./run-lambda-container-local.sh
```

# INVOKE!

```
./invoke-lambda-container.sh
```
